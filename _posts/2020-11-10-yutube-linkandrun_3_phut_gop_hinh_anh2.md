> ## **오늘의 단어**
>> **giữ gìn**: 지키다, 유지하다, 보관하다  
>> **đón**: 픽업하다, 납치하다  
>> **qua**: 건너다, (어디어) 들리다  
>> **mọi lần**: 매번  
>> **tránh ra**: 비켜  
>> **linh tinh**: 엉뚱한, 하찮은  
>> **xàm**: 엉망진창, 횡설수설  
>> **cao su**: 고무, 유연한  
>> **hôm nào**: 언젠가 (미래에 정해지지 않은)  
>> **tụ tập**: 모이다 (사람)  
>> **âm u**: 흐리다  
>> **vội**: 급히, 서두르다  
>> **yêu đương**: 연예, 연예하다  
>> **lại**: 또, 다가가다, 다시  
>> **tập trung**: 집중하다
