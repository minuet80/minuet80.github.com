> ## **오늘의 단어**
>> **cấm**: 금지, 금지하다  
>> **dừng**: 멈추다, 서다  
>> **đỗ xe**: 주차하다  
>> **hàng rong**: 행상  
>> **kinh doanh**: 장사하다  
>> **lấn chiếm**: 점거하다  
>> **vỉa hè**: 인도  
>> **lòng đường**: 도로  
>> **khu vực**: 구역  
>> **ô tô**: 자동차  
>> **được + 동사**: 동사하게 되다(수동태), (규정상) 동사 할 수 있다  
>> **đón trả**: (차/버스로 사람을) 태우고 내려주다  
>> **xin vui lờng**: ~해 주시겠습니까? (정중한 표현)  
>> **không + 서술어**: ~하지 않다  
>> **đi trên**: 표면위에 걷다, 다니다  
>> **cỏ**: 풀, 잔디  
>> **dẫm lên**: 밞다  
>> **trong nước**: 국내  
>> **quốc tế**: 국제  
>> **sảnh**: 로비  
>> **thủ tục**: 수속  
>> **khung đo**: 측정 틀  
>> **hành lý**: 짐, 수화물  
>> **xách tay**: 손으로 가지고 다니는  
>> **kiện**: 꾸러미  
>> **hạng**: 레벨, 등급  
>> **hành khách**: 승객  
>> **phổ thông**: 보통  
>> **giới hạn**: 제한  
>> **cài**: 매다  
>> **dây an toàn**: 안전벨트  
>> **trong khi**: ~하는 동안  
>> **áo phao**: 구명조끼  
>> **ghế ngồi**: 의자
